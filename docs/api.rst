API Documentation

This page details the api documentation for graphpro:

Graph Generation
----------------
A generator can be obtain from a specific library and return a graph generator.

    .. autofunction::
        graphpro.md_analisys


Then a graph generator can be used:


    .. automodule::
        graphpro.graphgen
